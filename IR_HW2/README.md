# IR_HW2

### Environment
- Django 2.2
- Python 3.10.0
- Docker

### Run website
- <h4>command</h4>
```
docker-compose up
```
- <h4>open website</h4>
```
127.0.0.1:5000/nlp/home
```
- <h4>clear web cache</h4>
```
Ctrl+F5
```
