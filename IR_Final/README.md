# IR_HW3

### Environment
- Django 2.2
- Python 3.10.0

### Run website
- <h4>command</h4>
```
pip install -r requirements.txt
python manage.py runserver 0.0.0.0:5000
```
- <h4>open website</h4>
```
127.0.0.1:5000/nlp/home
```
